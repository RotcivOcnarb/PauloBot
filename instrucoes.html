Comandos


paulo, criar comando [NOME] quando mensagem for do [PESSOA], canal [CANAL], mensagem comeca com "[TEXTO]", termina com "[TEXTO]", contem "[TEXTO1];[TEXTO2];[TEXTO3]", diga "[TEXTO1];[TEXTO2];[TEXTO3]" no canal [CANAL], incremente [VARIAVEL] com [VALOR], sobrescreva [VARIAVEL] com [VALOR], multiplique [VARIAVEL] com [VALOR]
[NOME] no caso pode ser tanto o ID quanto o nome da pessoa (recomendável ser o ID, visto que o nome pode ser mudado)

keywords:


${any} -> tanto pra pessoa quanto pra canal, significa qualquer um, utilizando no começo do comando. Ex:

paulo, criar comando [NOME] quando mensagem for do ${any}, canal ${any}

${channel} -> o canal que esse comando ativado
${user} -> o usuario que ativou o comando
${userID} -> o id do usuário que ativou o comando


Lembrando que todos os textos são case insensitive.

Na hora do filtro, quando colocar mensagem contem "TEXTO", você pode separar textos por ; nesse caso, ele vai checar se um dos textos coincide com a mensagem. Ex:

paulo, criar comando [NOME] quando mensagem for do ${any}, canal ${any}, mensagem contem "monark;mito"

este comando é disparado quando a mensagem contem tanto "monark" quanto "mito", ou os dois
Ao escrever diga "[TEXTO]", você pode separar por ; textos diferentes, nesse caso o bot vai escolher um desses textos aleatóriamente para dize-lo

Variaveis


É possivel também criar e utilizar variaveis usando o comando:

paulo, criar variavel [NOME] com valor [VALOR]

o bot vai criar uma variavel com o valor definido, que pode ser referenciada em comandos.

Quando se quer usar o valor de uma variável dentro de um texto, encapsule o nome dela com ${}, Ex.

... diga "hoje eu joguei lol ${numlol} vezes" no canal ${channel} ...

lembre-se que só se encapsula nos textos, quando referencia ela num comando, coloque o nome dela normalmente. Ex.

paulo, criar comando aumenta quanto mensagem for do ${any}, canal ${any}, mensagem contem "lolzinho", incremente numlol com 1

paulo, criar comando mostralol quando mensagem for do ${any}, canal ${any}, mensagem contem "mostra lol ae", diga "hoje eu joguei lol ${numlol} vezes" no canal ${channel}

Comandos pré feitos


existem alguns comandos básicos que já existem para fazer coisas que comandos não conseguem fazer normalmente:

paulo, qual meu id? -> Mostra o seu ID
paulo, qual meu nivel? -> mostra seu nivel atual
paulo, qual id desse canal? -> mostra o id do canal
paulo, deletar funcao [FUNCAO] -> deleta a função especificada
paulo, deletar variavel [VARIAVEL] -> deleta a variavel especificada
paulo, listar comandos -> lista todos os comandos criados (exceto os pré-feitos)
paulo, listar variaveis -> lista todas as variaveis criadas
paulo, mostrar [VARIAVEL] -> mostra o valor atual de uma variavel especificada
paulo, descrever [FUNCAO] -> mostra o JSON de uma função que especifica como ela funciona
